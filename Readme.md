# EduPress – School Portal WordPress Theme

EduPress is a modern Full Site Editing (FSE) WordPress theme designed for schools, learning platforms, and educational institutions.

## Features

- Full Site Editing support
- Custom block patterns (Hero, Features, Announcements)
- Responsive layout
- Built with theme.json design system
- Dynamic content using Query Loop

## Tech Stack

- WordPress Block Themes
- Gutenberg Editor
- theme.json
- Block Patterns

## Installation

1. Download the theme
2. Upload to `/wp-content/themes/`
3. Activate in WordPress dashboard
